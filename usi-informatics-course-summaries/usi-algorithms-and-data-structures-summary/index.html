Course overview

Design and analysis of classic algorithms; asymptotic complexity, core data structures and graph processing patterns.

Main topics

  • Asymptotic notation, recurrences & correctness proofs
  • Sorting & selection (Insertion, Merge, Quick, Heap, k-order statistics)
  • Fundamental data structures (hash tables, BSTs, red–black trees, heaps)
  • Graph algorithms (BFS/DFS, SCC, Dijkstra, Kruskal, MST)
  • Greedy & dynamic-programming paradigms + NP-completeness primer
