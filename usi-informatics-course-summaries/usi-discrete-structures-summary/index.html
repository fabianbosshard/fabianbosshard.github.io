Download the Discrete Structures Summary (PDF).

Course overview

A tour of the combinatorial and logical foundations of computer science: propositional reasoning, axiomatic set theory, counting techniques and an extended module on graph theory.

Main topics

  • Propositional logic — syntax, semantics, proof systems, resolution
  • Set theory & relations (ZFC axioms, equivalence & order relations)
  • Counting & combinatorics (binomial coefficients, inclusion–exclusion, pigeonhole)
  • Graph theory — trees, Euler/Hamilton cycles, planar graphs, colourings
